# InnoSoc AI Studio

**AI 工作室统一工作流前端界面** — 覆盖 AI 生图、图片编辑、视频生成、数字人视频生成与本地翻译能力。

---

## 项目概述

InnoSoc AI Studio 是一款面向企业级用户的 AI 创作平台，提供统一的工具入口和完整的内容生产工作流。平台整合了四大核心 AI 工具模块，覆盖从概念设计、图像编辑、视频生成到数字人制作的完整创作链路，同时接入本地大语言模型翻译工具，支持多语言内容生产。

### 核心价值

- **统一入口**：一个平台整合所有 AI 创作工具，降低学习成本
- **高效生产**：通过自然语言交互，快速生成专业级内容
- **企业级支持**：支持私有化部署、权限控制和审计能力
- **多端访问**：支持 Web、Windows、macOS 等多平台

## 核心工具介绍

### 1. 图像生成器（Image Generator）

**工具参数**: `tool=1`

**功能定位**：通过提示词、草图和参考图的组合，快速生成可用于设计提案的概念视觉图。

**核心能力**：
- 参考图驱动生成：基于提供的参考图像进行风格迁移和创意延伸
- 构图约束控制：支持对画面构图、元素位置进行精确控制
- 批量概念图生成：一次性生成多个版本供方案比稿

**适用场景**：
- 广告创意方案视觉提案
- 产品包装设计概念探索
- 品牌视觉方向比稿
- 游戏/影视概念设计

**业务价值**：将传统需要数小时的概念图制作时间缩短至分钟级，大幅提升设计沟通效率。

---

### 2. 图像编辑器（Image Editor）

**工具参数**: `tool=2`

**功能定位**：使用自然语言指令完成图像编辑，支持单图修复、双图融合、多图组合和局部重绘。

**核心能力**：
- 单图智能修复：对图像进行去瑕疵、补全、扩展等编辑
- 多图融合：将多张图像的元素智能融合为一张新图
- 局部重绘：对图像指定区域进行重新绘制或替换
- 自然语言交互：用文字描述即可完成复杂编辑操作

**适用场景**：
- 电商商品图优化（如灰模图上色、背景替换）
- 营销海报多版本快速改版
- 品牌视觉素材统一风格处理
- 人像精修与场景合成

**业务价值**：降低图像处理门槛，非专业设计人员也能完成高质量的图像编辑工作。

---

### 3. 首尾帧视频生成器（Frame-to-Frame Video Generator）

**工具参数**: `tool=3`

**功能定位**：从提示词或素材出发，快速生成可用于演示与传播的短视频内容。

**核心能力**：
- 镜头运动控制：精确控制视频的运镜方式和视角变化
- 时长调节：支持生成不同时长的视频片段
- 风格化视频生成：支持多种艺术风格的视频转换
- 首尾帧指定：通过首尾关键帧控制视频内容走向

**适用场景**：
- 社交媒体短视频创作
- 产品演示动画制作
- 活动预告视频生成
- 创意视觉动态展示

**业务价值**：将视频制作门槛大幅降低，无需专业视频编辑技能即可生成高质量动态内容。

---

### 4. 数字人视频生成器（Digital Human Video Generator）

**工具参数**: `tool=4`

**功能定位**：上传角色图像与音频即可生成口型驱动的数字人视频，适合讲解、客服与培训场景。

**核心能力**：
- 音频驱动口型：自动分析音频生成匹配的口型动画
- 动作脚本控制：支持预设和自定义数字人动作
- 自动合成输出：一键生成完整的数字人视频
- 多角色支持：可创建和使用多个数字人形象

**适用场景**：
- 产品讲解视频制作
- 客服应答视频生成
- 企业培训课程录制
- 品牌虚拟代言人内容

**业务价值**：无需真人出镜和拍摄，即可批量生成专业讲解视频，大幅降低视频制作成本。

## 其他工具

### 本地大语言模型翻译工具（Local Translate）

**访问地址**: https://fy.innosoc.com/

**功能定位**：本地化多模态翻译服务，接入 LM Studio 大语言模型。

**核心能力**：
- 文本翻译：支持多语言文本互译
- 图片翻译：识别并翻译图片中的文字内容
- 模型设置管理：支持自定义模型配置

**技术特点**：
- 本地模型接入：适合内网和数据敏感场景
- 隐私保护：数据无需上传至云端

**适用场景**：
- 国际化文案翻译处理
- 多语言素材整理
- 团队翻译工作流

## 快速开始

### 访问方式

所有工具模块通过统一入口访问：

```
http://localhost:8502
```

### 直达工具模块

可通过 URL 参数直接访问指定工具：

```
http://localhost:8502?tool=1  # 图像生成器
http://localhost:8502?tool=2  # 图像编辑器
http://localhost:8502?tool=3  # 首尾帧视频生成器
http://localhost:8502?tool=4  # 数字人视频生成器
```
